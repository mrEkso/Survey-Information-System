## Installation

1.копіюємо репозиторій

```sh
$ git clone https://gitlab.com/david.okayach/online-survey-system.git
```

2.розгортаємо проєкт

```sh
$ make up
```

3.
```sh
готово :)
```

### Додатково

- зупинити розгорнутий проєкт
```sh
  $ make stop
  ```
- видалити контейнери проєкта
```sh
  $ make down
  ```
- перезапустити проєкт
```sh
  $ make restart
  ```
